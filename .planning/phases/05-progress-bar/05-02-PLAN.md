    ```javascript

    ```

    REPLACE WITH:
    ```javascript
    // Start progress bar
    const progress = new ProgressReporter();
    progress.start(npcTokens.length, game.i18n.format("NPC_REPLACER.ProgressStart", { count: npcTokens.length }));
    ```

    **3. Add progress.update() inside the token processing loop (after the switch/case block, around line 1233):**

    ```javascript
    progress.update(replaced + notFound.length + importFailed.length + creationFailed.length,

        current: replaced + notFound.length + importFailed.length + creationFailed.length,
        total: npcTokens.length,
        name: tokenDoc.name
      }));
    ```

    Note: Use the sum of ALL result categories (not just `replaced`) as the `current` count, because every processed token contributes to progress regardless of outcome.

    **4. Add progress.finish() after the for loop, before #reportResults (around line 1236):**
    ```javascript
    progress.finish();
    ```

    This goes BEFORE `#reportResults()` so the progress bar dismisses and then the summary notification appears.
